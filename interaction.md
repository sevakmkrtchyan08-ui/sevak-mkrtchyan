# Interactive Resume Website - Interaction Design

## Core Interactive Components

### 1. Professional Timeline Navigator
- **Function**: Interactive career timeline with hover effects and detailed experience cards
- **Interaction**: Click on timeline points to reveal detailed job descriptions, achievements, and skills used
- **Visual**: Animated timeline with smooth transitions between experience periods
- **Data**: California Wellness Retreat (Oct 2023-Jun 2024), A1 Unlimited Home Health Care, CVHS Tech Team

### 2. Skills Proficiency Radar Chart
- **Function**: Dynamic radar chart showing technical and soft skills proficiency
- **Interaction**: Hover over skill categories to see detailed breakdown and experience level
- **Visual**: Animated radar chart with color-coded skill categories
- **Data**: Customer Service, Technology Troubleshooting, Social Media Management, Graphic Design, Time Management, Communication

### 3. Language Proficiency Display
- **Function**: Interactive language skills showcase with fluency indicators
- **Interaction**: Click on language cards to see proficiency details and usage contexts
- **Visual**: Animated progress bars with native/fluent indicators
- **Data**: English (Fluent), Armenian (Fluent)

### 4. Project Portfolio Gallery
- **Function**: Grid-based project showcase with filtering capabilities
- **Interaction**: Filter by skill type (Web Development, IT Support, Customer Service, Design)
- **Visual**: Masonry-style grid with hover effects revealing project details
- **Data**: Website development projects, IT support initiatives, social media campaigns

## Multi-Turn Interaction Flow

### Timeline Experience Explorer
1. User sees horizontal timeline with career milestones
2. Clicking a timeline point expands detailed experience card
3. Each card shows: role, duration, key achievements, skills utilized
4. Navigation arrows allow sequential exploration of experiences
5. "Learn More" buttons link to detailed project descriptions

### Skills Assessment Interactive
1. Radar chart displays initial skill overview
2. Hovering reveals specific competency levels (Beginner/Intermediate/Advanced)
3. Clicking skill categories shows related work experiences
4. Progress indicators animate on scroll into view
5. Skills filter experience timeline below

### Project Portfolio Browser
1. Grid displays all projects with category filters
2. Filter buttons: All, Web Dev, IT Support, Customer Service, Design
3. Hover effects reveal project descriptions and technologies used
4. Click opens detailed project modal with images and outcomes
5. Related projects suggested at bottom of modal

## User Journey Design
- **Landing**: Hero section with animated introduction and key highlights
- **Explore**: Interactive timeline encourages deep dive into experiences
- **Skills**: Visual representation of capabilities with interactive elements
- **Portfolio**: Filtered project showcase demonstrating practical application
- **Connect**: Contact form with availability status and preferred communication methods

## Accessibility & Usability
- Keyboard navigation support for all interactive elements
- Screen reader friendly descriptions and alt text
- Mobile-responsive touch interactions
- Loading states and smooth transitions between sections
- Clear visual hierarchy and intuitive navigation patterns
